# Contact Form with JavaScript Validation

This project is a simple Contact Form built using **HTML, CSS, and JavaScript**. It demonstrates client-side form validation by checking that the Name, Email, and Message fields are completed before submission. Email addresses are validated using a regular expression (Regex), and clear error messages are displayed below each field when validation fails. On successful validation, the form displays a success message without sending any data to a server.

## Features

* Responsive contact form
* Client-side input validation
* Name, Email, and Message validation
* Email format validation using Regex
* Error messages displayed below inputs
* Prevents form submission when inputs are invalid
* Success message on valid submission
* Clean and user-friendly interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Learning Outcomes

* Form handling in JavaScript
* Client-side validation techniques
* Using Regular Expressions (Regex)
* DOM manipulation
* Providing user-friendly feedback
